# 👥 Previsão de Turnover

> Previsão de rotatividade de funcionários com ML explicável. Identifica quem está em risco de sair e recomenda ações de retenção.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/previsao-turnover/

## 📊 Resultados-chave (KPIs)
**0.91** AUC · **+24%** Retenção · **3 meses** Antecip. · **R$ 8M** Custo evit.

## 🧠 Tecnologias de IA
ML · SHAP · People Analytics

## ✨ O que faz
- Score de turnover por funcionário
- Top fatores de risco (SHAP)
- Ação de retenção recomendada
- ROI de retenção

## 🗄️ Fonte de dados real
[RH + engajamento](https://www.gov.br/trabalho)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com RH + engajamento
2. **Processamento com IA** — ML · SHAP · People Analytics
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
